## Run Locally

Clone the project

```bash
  git clone https://github.com/SarahNaif/PEAN-Ecommerce.git
```

Go to the project directory

```bash
  cd PEAN-Ecommerce
```

#### server/.env

POSTGRES_DB_TEST
